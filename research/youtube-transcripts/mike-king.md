## Key Insights Extracted

**On why traditional SEO is deprecated:** Most SEO practices were built for lexical search (counting keywords). Google has been semantic (understanding meaning) since 2013. Almost all SEO software is still lexical. This gap explains inconsistent results from following best practices.

**On Relevance Engineering:** Combines information retrieval, AI, content strategy, UX, and digital PR. Understanding how systems actually work produces more consistent results than following checklists.

**On RAG:** AI search systems extract passages and recombine them. Every section of a page should be independently coherent and directly answerable as a standalone passage.

**On the 499 response code:** AI platforms fetch pages in real time. If a page loads too slowly, the system issues a 499 code and moves on. Page speed is now an AI search eligibility requirement.

**On metadata:** In AI search, ChatGPT evaluates page title, meta description, and URL slug to determine whether to even fetch the full page. Metadata that matches the user's prompt significantly increases inclusion probability.
