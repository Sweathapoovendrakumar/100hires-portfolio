# AI Tools Setup – Portfolio Project

## Tools Installed

- **Cursor IDE** – AI-powered code editor (cursor.com)
- **Claude Code** – AI coding assistant extension inside Cursor
- **Codex** – AI code generation extension inside Cursor

## Steps Completed

1. Created a GitHub account and set up a public repository
2. Downloaded and installed Cursor IDE
3. Searched for and installed the Claude Code extension within Cursor
4. Searched for and installed the Codex extension within Cursor
5. Created this README.md file to document the process

## Challenges & How I Solved Them

- Initially opened Windsurf instead of Cursor — realized the interface looked different, searched specifically for Cursor at cursor.com and opened the correct tool
- Git was not recognized in Cursor's terminal — installed Git separately from git-scm.com and added it to system environment variables
- Switched to creating the README directly on GitHub to avoid terminal issues

## Reflection

This process introduced me to version control workflows and AI-assisted development environments. Being able to troubleshoot independently and find solutions quickly is a skill I am actively building, and this task was a practical way to demonstrate exactly that.
